# Pear Tree Well Website

A React/Vite application for Pear Tree Well mineral water company.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Contact Form

The contact form uses Netlify's built-in form handling service. When deployed to Netlify, form submissions will be processed automatically and emails will be sent to the configured notification email in Netlify's dashboard.
